#	Quiz 
A Quiz game built in 2017 using NodeJS and TingoDB.

##	Installation
```
	git clone https://github.com/ianpuchetti/quiz.git
	cd quiz
	npm install
	npm start
```
Then, open http://localhost:3000 on your browser
